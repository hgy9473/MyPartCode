CSS 3 text-shadow 试验

试验 1

CSS 源码


.shadow01{
	background: white;
	font-size: 3em;
	font-weight: bold;
	text-shadow: 2px 2px  4px #000;
	padding:10px;
	font-family: "宋体","楷体";
}
