CSS3 box-shadow 试验

语法：box-shadow: h-shadow(水平移动) v-shadow(垂直移动) blur(模糊距离) spread(阴影大小) color inset（或outset）;

试验 1

CSS 源码


.shadow01{
	width:600px;
	box-shadow: 5px 5px 3px #333;
	background:white;
}

效果

什么是 AJAX ？ AJAX = 异步 JavaScript 和 XML。 AJAX 是一种用于创建快速动态网页的技术。 通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。 传统的网页（不使用 AJAX）如果需要更新内容，必需重载整个网页面。
