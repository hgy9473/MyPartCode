CSS :after试验

试验1 -- 添加文字

源码


/* 简单的:after样式 */
.boring-text::after {
    content:    "<- 无聊!";
    color:      red;
}

效果

天若有情天亦老

人间正道是沧桑

试验2 -- 创建一个大空间

源码


/* 复杂的:after样式 */
.exciting-text::after {
    content: "<- 忍俊不禁!"; 
    color: green;
    display: block;
    background:pink;
    height:100px;
    font-size:2em;
    border-radius: 20px;
    border:2px solid red;
    box-shadow: 5px 5px 4px #333;
}

效果

匹夫无罪怀璧其罪

试验3 -- 配合hover实现文字提示

源码


/* after 联合 hover */
span[data-descr] {
    position: relative;
    font-family: 楷体;
    font-size:1.2em;
    font-weight: bold;
    cursor: help;
}

span[data-descr]:hover::after {
    content: attr(data-descr);
    position: absolute;
    left: 0;
    top: 24px;
    min-width: 10px;
    text-align: center;
    border: 1px #aaaaaa solid;
    border-radius: 10px;
    background-color: #ffffcc;
    padding: 12px;
    color: #000000;
    font-size: 24px;
    z-index: 10;
}

效果

云横秦岭家何在

雪拥蓝关马不前
